FAQ

Common questions


Is AppliedIn completely free?
Yes. Free forever. No subscription, no premium plan, no hidden charges. Download from GitHub and use it forever at zero cost.
Which job portals are supported?
LinkedIn, Naukri, Internshala, Indeed, Glassdoor, Unstop — all automatic. Plus any company career website like TCS, Infosys, Wipro, Deloitte, Accenture, and more.
Is my data safe? Who can see my applications?
Nobody. All data is stored only on your own device using Chrome local storage. No server involved. Not even the developer can see your applications.
How do I install it without the Chrome Web Store?
Download the ZIP from GitHub → Open chrome://extensions → Turn on Developer Mode → Click Load Unpacked → Select the extension folder. Done in under 2 minutes.
What if I apply on a company's own website — like TCS or Infosys?
AppliedIn detects apply button clicks on any website. For company career pages, a small confirmation popup appears asking "Did you complete this application?" — tap Yes and it's saved.
Can I export my applications to Excel?
Yes. Export to Excel (.xlsx) or CSV anytime. Choose by time period — Today, This Week, This Month, This Year, or All Time. Your data, your format, offline.
